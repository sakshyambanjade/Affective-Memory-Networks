\section{Related Work}

\subsection{Cognitive Science of Memory}
Emotional memories show superior recall via amygdala tagging \cite{mcgauch2004amygdala}. Peak-end bias organizes experience weighting \cite{kahneman1999evaluative}. Lazarus appraisal theory structures emotional significance \cite{lazarus1991emotion}.

\subsection{LLM Memory Systems}
Semantic RAG \cite{lewis2020retrieval} organizes by topical similarity. Recency baselines \cite{medjedovic2023llm}. Reflexion \cite{shinn2023reflexion} uses self-reflection but lacks emotional structure.
