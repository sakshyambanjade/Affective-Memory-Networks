\title{Emotional Memory Improves Long-Context Coherence in LLM Agents}

\abstract{
LLMs lose coherence over long conversations. We hypothesize emotional memory—human-like organization by VAD dimensions—improves continuity. Our Affective Memory Network (AMN) implements appraisal $\rightarrow$ working memory (cap=5) $\rightarrow$ episodic consolidation $\rightarrow$ emotionally-weighted retrieval (complementary resonance). 

\textbf{Results:} Across 30×50-turn conversations, AMN achieves 42\% memory reference rate vs 8\% baseline (p$<$0.001), 28\% coherence gain, 34\% emotional appropriateness gain. First evidence emotional architectures measurably improve agent continuity.
}
